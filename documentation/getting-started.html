Getting Started

What is Vue?

Vue is a theme based on Bootstrap 3. and inspired by Google's Material Design

Vue tightly integrates with Material Design to bring your app's landing page a look and feel that is far ahead of the current web design standards. Material Design is the next big trend in web design and Vue is the first to bring you it.

How to Use It?

Since Vue is built on top of Bootstrap as a theme, you can use it in your Bootstrap 3 project. We did not modify any line of Bootstrap, so you will be safe using Vue in your ongoing project.

To edit the look and feel of Vue, all you need to do is edit the CSS sources.
1. All the major CSS styles are housed in the style.css stylesheet, located in - "static/css/style.css". For most changes this is where you'll need to go in order to make changes.

2. The easiest way to start working with Vue on your site is to simply take the main index.html and edit it to suit your needs.

File Structure

Once purchased, unzip the compressed archive to see the following structure: 

  ├── documentation/
    ├── resources/
        ├── cards/
        ├── fonts/
        ├── mockups/
  ├── static/
      ├── css/
      ├── fonts/
      └── icons/
      ├── img/
      ├── js/
      └── less/
      ├── favicon.png
  ├── index.html

Let's go through this tree list:

documentation/ As you can see this directory contains all documentation for Vue. In this directory the two main files that you should open are getting-started.html (Which is this page) and components.html. These two pages will take you through all the documentation that you need to know about Vue.
Take note to pay special attention to components.html, which gives you a breakdown of all the individual components that make up Vue.
.

resources/ The resources directory contains all external resources that are not used by the website. Within the resources directory you will find all mockups and resources which you can use to further edit your site.

Take special note of the mockups/ folder which contains all the 20+ Smartphone .PSD Mockups which you can use to display your App within on your website.

static/ This directory contains all the static resources needed to load index.html. All resources that are currently used by the website can be located in sub-directories of this directory.

Be careful not to delete any files from here unless you are absolutely sure as this could result in your website not displaying/loading correctly.
